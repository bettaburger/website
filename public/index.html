Projects

Most of my projects were created to help speed up tasks in the terminal and were developed using Golang and libraries such as Cobra and Tview.


The Chimera Dashboard was my first project working with clients. It was also my first experience touching and querying existing data from servers with Prometheus and then visualizing them using Grafana. $netstat and $telnet was particularily helpful for this project. Searching up processes started to become annoying and so I created Pobby to help me.


Initially, Pobby was developed using Bubbletea, but I found the ELM architecture complicated to work with as I added more functionality.
